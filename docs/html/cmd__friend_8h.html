Detailed Description

CMD_Friend - The Command line friend library!

This library is made for C and C++ programs to parse command line options in the easiest way possible. Based heavily on the argp library of the GNU project.

Created by: Jo√£o Peterson Scheffer - 2020.

This is the header file, please don't modify, except when stated in documentation.

Refer to this header as means to use the library and get informations about the variables and functions.

typedef int(* option_parse_function) (char key, char *arg, int arg_pos, void *extern_user_variables_struct)

Typedef from user parse option function. The "cdmf_parse_options" function will call the user defined function to handle actions as desired. Each call will have a key and argument, where each argument must have be assigned to only one key.

Parameters
keyReceives the char key of the option.
argReceives the argument passed to that option.
arg_posThe command line argument index where the user inserted the argumment.
arg_posOpaque pointer to user define struct in main program, containing variables that can be acesses inside this function. (Cast is necessary!).

option_parse_function  parse_function,

char **  argv,
cdmf_PARSER_FLAGS_Typedef  flags,
void *  extern_user_variables_struct 
)

Main library function, used to parse options in main program. The "cdmf_parse_options" function will call the user defined function to handle actions as desired. Each call will have a key and argument, where each argument must have be assigned to only one key.

Parameters
registered_optionsReceives pointer to struct "cmdf_options" array that shall contain user define options.
parse_functionUser defined parse function pointer.
argcMain function parameter containing number of passed parameters in command line.
argvMain function parameter containing the array of string containing the actual parameters.
flagsFlags used to customize the function "cdmf_parse_options" behavior, flags shall be located on "cdmf_PARSER_FLAGS_Typedef" enumerator.
extern_user_variables_structOpaque pointer to user define struct in main program, used to be accessed in the also user define parser function.
Returns
Returns a integer number correponding to a error code, 0 if no error occurred. Note: NO ERROR CODE IMPLEMENTED AT THIS VERSION.
